# 🏛️ Générateur de Galerie d'Art (The MET API)

Ce projet est une application Python qui automatise
la création d'une galerie d'art web à partir des données 
publiques du **Metropolitan Museum of Art** de New York.


## 📋 Fonctionnement du projet

Le projet est divisé en deux scripts pour séparer 
la logique de récupération et la logique d'affichage :
`2_europeanPaintings.py` : qui récupère les données grâce 
à l'API du musée.
`3_site.py` : qui affiche les oeuvres.
