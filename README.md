# Portfolio

Bienvenue sur mon portfolio ! 👋

Ce repository a pour but de centraliser l’ensemble de mes projets informatiques, académiques et personnels.

Le but est de documenter ma progression et de regrouper mes différents travaux au même endroit. Ce portfolio va grandir et évoluer au fil du temps avec l'ajout de nouveaux projets.

## 📂 Organisation
Les projets sont classés par dossiers thématiques. Pour faciliter la navigation, chaque projet (sous-dossier) possède son propre fichier `README.md` qui détaille le contexte, les objectifs et les résultats de l'exercice.
