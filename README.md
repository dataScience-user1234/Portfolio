> [!TIP]

> **Évolution de mon parcours**

> [!WARNING]
> **📂 Organisation**
> Les projets sont classés par dossiers thématiques. Pour faciliter la navigation, chaque projet possède son propre fichier `README.md` qui détaille le contexte, les objectifs et les résultats.
