Petit projet Python pour visualiser l'évolution des températures à Aurillac de 1993 à 2023 en utilisant l'API d'open meteo.

1.  J'ai récupèré les données météo d'Aurillac via l'API **Open-Meteo**.
2.  Puis j'ai calcule la moyenne annuelle pour 5 année (1983, 1993, 2003, 2013, 2023) avec le module `statistics`.
3.  Enfin j'ai génèré une courbe d'évolution (graphique_aurillac.png) avec `matplotlib` pour bien montrer l'évolution.
