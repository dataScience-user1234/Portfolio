Ce projet est une architecture client-serveur complète comprenant :
- Une **API RESTful** codée en Python avec FastAPI.
- Une **base de données relationnelle** gérée via SQLModel et SQLite (`database.db`).
- Un **Client Web** (Single Page Application) en HTML, CSS et JavaScript.

L'application permet aux utilisateurs de s'authentifier (via JWT), de créer une armée, de recruter des chats et de les équiper d'armes.


**Installation et Démarrage**

1. **Créer l'environnement virtuel :**
python3 -m venv .venv

2. **Activer l'environnement virtuel :**
source .venv/bin/activate

3. **Installer les dépendances :**
pip install -r requirements.txt

4. **Lancer le serveur :**
fastapi dev app.py

5. **Ouvrir le client web :**
Clique droit sur `index.html` puis sélectionnez Open with Live Server.

La documentation générée automatiquement : 
http://127.0.0.1:8000/docs (générée via Swagger UI)
