Ce dépôt regroupe deux projets, qui explorent la communication entre un client et un serveur à travers deux exercices distincts. Le premier dossier contient une application web (frontend) qui interagit de manière asynchrone avec l'API publique PokéAPI. Le second dossier abrite "La Guerre des Chats", une architecture client-serveur complète comprenant la création d'une API RESTful sécurisée avec FastAPI, sa base de données locale et son interface web. Vous trouverez les instructions de démarrage spécifiques à chaque exercice dans le fichier README de leur dossier respectif.
